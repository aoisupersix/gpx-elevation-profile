# Development Guidelines

A web application that generates color-coded elevation profiles from GPX files according to gradient.

## Coding conventions

- File names: kebab-case (`example-file-name.ts`)
- Write comments, documentation comments (JSDoc) and git commit messages in English
- Write documentation (README.md, etc.) in English

## Workflow

- Create a topic branch for each unit of work and merge it into `master` via a pull request
